Quick Start

1

Install OpenMCP

pip install openmcp
2

Initialize Configuration

openmcp init-config
3

Start the Server

openmcp serve
4

Use in Your Code

import openmcp

async with openmcp.browser() as browser:
    await browser.navigate("https://example.com")
    await browser.screenshot("page.png")
